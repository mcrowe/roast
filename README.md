# babel-plugin-react-cl

Add classes to your JSX by writing `cl`, and save the `assName`.

## Installation & Usage

Install the plugin:

```
# For Babel 6
npm install --save-dev babel-plugin-react-cl
```

Then edit your `.babelrc` to include `react-cl`:

```json
{
  "plugins": [
    "react-cl"
  ]
}
```
